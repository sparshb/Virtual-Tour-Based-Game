Virtual Tour based Game


The software is a virtual tour based game. The game has 2 modes: 'Play Tour' and 'Start Game'. 
In Tour mode, the user is allowed to roam freely through the map. A label is shown when the user 
is in the vicinity of a room.
In Game mode, the user is spawned at a random location in the map and is asked to reach a destination 
within 60 seconds. The user is awarded score on completion of the task and 0 if the task is not completed.
There is also option to alter the music volume.
